# Personal Portfolio - Jaymin West

A modern, responsive portfolio website built with React, TypeScript, and Tailwind CSS. Features an animated background, interactive sections, and a clean, minimalist design.

## Features

- Animated gradient background transitions
- Responsive design for all screen sizes
- Interactive hero section with rotating titles
- Project showcase with live links
- Services overview
- About me section with skills and interests
- Contact integration with email copy functionality
- Smooth scrolling navigation
- Accessibility-focused components

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- Vite
- Lucide React Icons

## Getting Started

1. Clone the repository
2. Install dependencies:
```
npm install
```

3. Create a `.env` file in the root directory and add your OpenAI API key (optional):
```
VITE_OPENAI_API_KEY=your_api_key_here
```

4. Start the development server:
```
npm run dev
```

5. Build for production:
```
npm run build
```

## Project Structure

- `/src/components`: React components
- `/src/lib`: Utility functions and types
- `/public`: Static assets

## Key Components

- AnimatedBackground: Handles the gradient background transitions
- Hero: Main landing section with rotating titles
- Companies: Showcase of current projects
- Services: Overview of professional services
- AboutMe: Detailed background and skills section
